## Adding an Entry

- **ID**: `DEBT-001`, `DEBT-002`, etc.
- **Area**: Domain this affects (e.g., `research/competitors`, `docs/architecture`)
- **Description**: What is missing or wrong
- **Impact**: H / M / L — how much this impairs agent effectiveness
- **Added**: Date first logged

When debt is resolved, move the row to the Resolved section below with a resolution note.
